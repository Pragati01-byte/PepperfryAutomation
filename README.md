# 🛒 Pepperfry Automation Project

## 📌 Project Overview
This project is a Selenium automation framework developed to test basic functionality of the Pepperfry website.

The main objective of this project is to:
- Open the Pepperfry website
- Handle popup if present
- Navigate to Cart page
- Fetch product details from the cart
- Validate cart functionality using TestNG assertions

---

## 🛠️ Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- IntelliJ IDEA
